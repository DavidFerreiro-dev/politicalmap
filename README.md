# politicalmap
Proyecto web interactivo que visualiza el mapa político de España (y mas territorios) en un diagrama de Nolan. Posiciona a los partidos políticos en dos ejes: Izquierda-Derecha y Comunitarista-Liberal. La interfaz incluye filtros por estado de representación, histórico de formaciones, buscador integrado y soporte para analizar otros países.
